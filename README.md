# Slope 2D 🟢

A fast-paced browser dodge game. Avoid the red balls, collect the orbs, and survive as long as you can.

## 🎮 Play it

Open `index.html` in any modern browser — no install, no dependencies.

**Or host it free on GitHub Pages:**
1. Fork / upload this repo to GitHub
2. Go to **Settings → Pages → Source → main branch / root**
3. GitHub gives you a live URL in ~60 seconds

## Controls

| Key | Action |
|-----|--------|
| `W A S D` | Move the player |
| `Enter` | Start / Restart |
| `Shift` | Return to Main Menu |

## Features

- 🌊 **Wave system** — speed ramps up every 10 points
- 💫 **Rare orbs** — Gold (common) → Purple → Pink → 🌈 Rainbow (1% chance, 100 pts!)
- ⚡ **FPS setting** — 5 / 30 / 45 / 60 FPS (including Slow-Mo Matrix Mode)
- 🎨 **Background colours** — pick your vibe in Settings
- 🏆 **High score** — tracked per session

## Files

```
slope2d/
└── index.html   ← entire game, self-contained
```

Everything is in one file — pure HTML, CSS, and JavaScript. No frameworks, no build step.

## How it was built

Originally written in Code.org App Lab, then ported to a standalone web game using the HTML5 Canvas API.
